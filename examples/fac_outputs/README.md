# FAC Output Files

This folder contains output files generated by FAC (Flexible Atomic Code) calculations.

## Organization

- `fe24.lev.asc` - Energy levels for Fe XXIV autoionization calculation
- `fe24_ai.tr.asc` - Radiative transitions in autoionizing manifold
- `fe24_ai.ai.asc` - Autoionization rates
- `*.b` files - Binary FAC data files (used internally)

## Reading with AtomKit

```python
from atomkit.readers import read_fac, read_fac_transitions, read_fac_autoionization

# Read energy levels
levels = read_fac('examples/fac_outputs/fe24')

# Read radiative transitions
transitions = read_fac_transitions('examples/fac_outputs/fe24_ai')

# Read autoionization rates
ai_rates = read_fac_autoionization('examples/fac_outputs/fe24_ai')

print(f"Loaded {len(levels)} levels, {len(transitions)} transitions, {len(ai_rates)} AI rates")
```

## Generation

These files are generated by running:

```bash
cd examples/fac_inputs
sfac fe24_autoionization.sf
```

The input file is a simplified Fe XXIV autoionization calculation with 3 target + 4 autoionizing configurations.
